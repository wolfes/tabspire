Tabspire

A command line for your browser's address bar.

Name the tabs you want at your fingertips to re-open or re-focus later.

Easy Installation

  1. Download Tabspire.
  2. Open Chrome Extension Page --> chrome://chrome/extensions
  3. Drag tabspire.crx into Chrome Extensions Page.

Features

  • Add name 'api/twitter' to current tab:
    • t a api/twitter
  • Open or re-focus 'api/twitter' with fuzzy name search:
    • t o twit
    • t o apt
  • Message yourself later:
    • t m 14:30 You'll see this at 2:30pm.
    • t n 30 You'll see this in 30 minutes.
    • Just click the notification box to close it.
  • Reload a page every n seconds:
    • t r 60
  • Reload window's tabs (that optionally have urls matching param):
    • t rw
    • t rw yelp
  • Bookmarklets!
    • t b edit document.body.contentEditable=true;
    • t u edit --> Uses edit bookmarklet on current tab.
    • Supports fuzzy search for bookmarklet name.
